# Little Pizza House



## Project Description
Little Pizza House is a relaxing, stress-free cooking simulator for PC where you get to run your own restaurant. Players take their time preparing customer orders by baking pizzas with specific ingredients and serving the correct drinks. Earning money from perfect orders allows you to upgrade your menu items, like toppings and drinks, helping you grow your business at your own pace without the pressure of a countdown timer.

---

## Installation / Running Guide For Mac and Window

### Prerequisites
- Python 3.7 or higher
- Pygame library

### Installation Steps

**For Mac:**
1. Open Terminal
2. Navigate to the project directory:
3. Install required dependencies:
   ```
   pip install pygame
   ```
4. Run the game:
   ```
   python3 main.py
   ```

**For Windows:**
1. Open Command Prompt
2. Navigate to the project directory:
3. Install required dependencies:
   ```
   pip install pygame
   ```
4. Run the game:
   ```
   python main.py
   ```

## Tutorial / Usage
* Use comfortable mouse controls (clicking) to interact with the game.
* Check the randomized customer orders you receive.
* Select the correct ingredients to bake a pizza and prepare a drink that matches the order.
* Serve the completed meal to the customer perfectly to gain money.
* At the end of each day, use your earned money to purchase upgrades for your toppings and drinks to increase their value.

---

## Game Features
* **Stress-Free Environment:** Cook at your own pace with no fast-ticking timers.
* **Menu Management:** Create new pizzas, add/upgrade toppings, and add/upgrade drinks.
* **Business Progression:** Earn money for correct orders and watch your pizzeria grow.
* **In-game Statistics:** Continuously record and check order-based data (saved to a CSV file) tracking item prices, time taken per order, popular toppings/drinks, and order accuracy.
